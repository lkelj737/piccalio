## 一个从威尼斯海滩走出来的品牌

Piccalio这个牌子2019年在加州威尼斯起家,创始人Aneta Kostic当时怀着娃,到处找那种既好看又实用的儿童家具,发现市面上不是塑料感太重就是设计太幼稚。她想要的是那种可以融入家居环境、不会显得突兀的木质家具。

于是她干脆自己做了一个品牌。理念很简单:用天然材料做出既符合蒙台梭利教育理念,又不会破坏家里装修风格的儿童家具。

说白了,就是那种你摆在客厅不会觉得丑、孩子用起来又特别顺手的东西。

## 用户真实评价怎么样?

我看了好几个平台的评价,整体来说褒贬不一,但好评占大多数。

不过Piccalio的客服团队在评论下都有回复,态度还算诚恳,表示会改进。

整体来说,产品质量大家都认可,主要问题集中在物流和售后服务上。如果你不赶时间,这些问题应该不大。
